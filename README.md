## Description

Tool to add monthly watched movies and shows to Trakt lists.

## Installation

```bash
$ yarn install
```

Rename `.env.example` to `.env`

Set `TRAKT_CLIENT_ID` and `TRAKT_CLIENT_SECRET` - https://trakt.tv/oauth/applications

## Running the app

```bash
# Run
$ yarn start

# Build
$ yarn build
$ yarn start-build
```
